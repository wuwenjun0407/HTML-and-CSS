## 2.HTML的标签
```
#<h1></h1>
##<h2></h2>
###<h3></h3>
####<h4></h4>
#####<h5></h5>
######<h6></h6>
“#”与h标签相对应
标题标签<h1>-<h6>逐渐变小 
需要的标签例如<h1>，按下tab键，自动生成一组完整的标签。例如<h1></h1>
F12 打开浏览器的控制台
F5  刷新浏览器的页面
Elements 表示元素（HTML 里面的标签）
1.<h1> 标题标签 <h1>-<h6>
2.<p> 标签可以放文章或者段落。
3.<strong> 文字加粗。有情调的作用（可以放在文字的任意地方，但是要有开始标签和结束标签。）
                 例如：<p>这是<em>一</em>个<strong>段落</strong>
                      </p>
4.<em> 斜体 em和i 效果一样。有强调的作用
  <i>  斜体
5.<ul> 无序列表 （无数字） 下面只能放<li>标签
6.<ol> 有序列表 （有数字） 下面只能放<li>标签
7.<li> 列表内容文字  
                 例如：<ul>
                         <li></li> 
                          <li></li> 
                          <li></li> 
                      </ul>   
8.<div> 相当于盒子，里面放其他所有标签。 
                 例如:<div>
                        <h1></h1>
                        <p></p>
                        <strong></strong>
                        <ul></ul>
                        ...
                      </div>   
9.<img> 插入图片   src:图片端的路径地址（相对路径和绝对路径） alt:表示图
                  果不存在，有文字显示。
10.<br/> 换行 （自闭合标签，没有开始标签）
11.<a href="#"></a>  外部链接
   <a href="#" target="_blank"></a>
   在另一个页面打开
     <a href="#" target="_self"></a>
     在现在的页面打开
 12.<textarea cols="" rows="">
 13.<sup> 上标
 14.<>    下标
 15.<input>   
```
